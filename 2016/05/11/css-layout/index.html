css 布局

概念

  • display(所有的display)
    • block(块级)
    • inline(行级)
    • none(不显示)
    • 其他(inline-block,flex…)

一个块级元素会新开始一行并且尽可能撑满容器。其他常用的块级元素包括 p 、 form 和HTML5中的新元素： header 、 footer 、 section 等等

  • margin

设置块级元素的 width 可以阻止它从左到右撑满容器。然后你就可以设置左右外边距为 auto 来使其水平居中。元素会占据你所指定的宽度，然后剩余的宽度会一分为二成为左右外边距。

当浏览器窗口比元素的宽度还要窄的时候使用 max-width 替代 width可以使浏览器更好地处理小窗口的情况。这点在移动设备上显得尤为重要

  • position
    • static
    • relative
    • fixed
    • absolute
  • float
